SpawnProjectile

Spawn a projectile.

Returns

a pointer to the spawned projectile or NULL if the spawn was not successful.

Description

WorldContextObject

The blueprint of the projectile to spawn.

The team that owns the projectile.

The amount of damage the projectile will do.
