GiveBuff

Adds buff for specified strategy character.

Returns

true if the actors are on different teams.

References

Module

StrategyGame

Header

/StrategyGame/Source/StrategyGame/Public/StrategyGameBlueprintLibrary.h

Include

#include "StrategyGameBlueprintLibrary.h"

Syntax

static void GiveBuff
(
    AStrategyChar * InChar,
    int32 AttackMin,
    int32 AttackMax,
    int32 DamageReduction,
    int32 MaxHealthBonus,
    int32 HealthRegen,
    float Speed,
    float Duration,
    bool bInfiniteDuration,
    int32 AttackDistance
)

Remarks

Adds buff for specified strategy character.

Parameters

Parameter

Description

InChar

The strategy character to apply the buff values

AttackMin

Set the characters Minimal melee attack damage.

AttackMax

Sets the characters Maximum melee attack damage.

DamageReduction

Amount to reduce the characters damage by.

MaxHealthBonus

Amount to increase the characters health by.

HealthRegen

Amount to regenerate the characters by.

Speed

Amount to adjust the characters Movement speed by.

Duration

Duration for the values to apply.

bInfiniteDuration

Should this buff apply forever.

AttackDistance

Amount to increase the attack distance by.
